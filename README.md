## 脚本使用
### js版本
下载nodejs，教程看这里 -> [Node.js 安装配置](https://www.runoob.com/nodejs/nodejs-install-setup.html)
安装依赖：
```sh
npm install -g axios
```
运行 `node GLaDOS_Checkin.js`

### python版本
```sh
pip install requests
```
运行：
```sh
python check_in.py
```
